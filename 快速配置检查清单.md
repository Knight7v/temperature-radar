# 阿里云 OSS 快速配置检查清单

打印此页，按步骤操作：

---

## ☑️ 配置清单

### 账号准备
- [ ] 注册阿里云账号
- [ ] 完成实名认证
- [ ] 充值 10 元

### 创建 Bucket
- [ ] 搜索「OSS」进入控制台
- [ ] 创建 Bucket
- [ ] 名称：`temperature-radar-xxx`（自定义）
- [ ] 地域：华北2（北京）
- [ ] 读写权限：**公共读** ⚠️

### 开启静态网站托管
- [ ] 进入 Bucket → 数据管理 → 静态页面
- [ ] 点击「设置」
- [ ] 默认首页：`index.html`
- [ ] 默认 404 页：`index.html`
- [ ] 勾选「开通子目录首页」
- [ ] 保存

### 上传文件
- [ ] 进入文件管理
- [ ] 上传 `oss-deploy` 文件夹内容：
  - [ ] index.html
  - [ ] css/ 文件夹
  - [ ] js/ 文件夹
  - [ ] assets/ 文件夹

### 验证
- [ ] 访问静态网站托管地址
- [ ] 检查页面显示正常
- [ ] 检查地图显示正常
- [ ] 分享给团队

---

## 🔑 关键设置提醒

| 设置项 | 正确值 | 错误后果 |
|--------|--------|---------|
| 读写权限 | 公共读 | 403 错误 |
| 静态网站托管 | 开启 | 404 错误 |
| 默认首页 | index.html | 无法访问 |
