# Vercel 国内访问问题解决方案

## 问题说明
Vercel 部署完成但国内网络无法访问，这是因为 Vercel 的域名 (`*.vercel.app`) 在部分国内网络环境下存在访问限制。

---

## 🎯 推荐方案（按优先级）

### 方案 1：阿里云 OSS 静态托管 ⭐⭐⭐⭐⭐

**优势**：国内访问快、稳定、便宜（约1元/月）

**详细步骤**：

#### 1.1 创建 OSS Bucket
```
1. 登录 https://www.aliyun.com
2. 搜索 "OSS" 进入对象存储控制台
3. 创建 Bucket：
   - 名称: temperature-radar-xxx (自定义，全局唯一)
   - 地域: 华北2(北京)
   - 读写权限: 公共读 ⚠️
```

#### 1.2 开启静态网站托管
```
1. 进入 Bucket → 数据管理 → 静态页面
2. 点击"设置"
3. 配置：
   - 默认首页: index.html
   - 默认 404 页: index.html
   - 开通子目录首页: ✅
4. 保存后会显示访问地址
```

#### 1.3 上传文件
```bash
# 运行准备脚本
bash deploy-aliyun-oss.sh

# 然后在 OSS 控制台上传 oss-deploy 文件夹内容
```

#### 1.4 访问地址格式
```
http://temperature-radar-xxx.oss-cn-beijing.aliyuncs.com
```

### 方案 2：腾讯云 COS 静态托管 ⭐⭐⭐⭐

**详细步骤**：

#### 2.1 创建 COS 存储桶
```
1. 登录 https://cloud.tencent.com
2. 搜索 "COS" 进入对象存储控制台
3. 创建存储桶：
   - 名称: temperature-radar-xxx
   - 地域: 北京
   - 访问权限: 公共读
```

#### 2.2 开启静态网站
```
1. 进入存储桶 → 文件列表 → 静态网站
2. 点击"编辑"
3. 配置：
   - 索引文档: index.html
   - 错误文档: index.html
4. 保存
```

#### 2.3 访问地址格式
```
https://temperature-radar-xxx.cos.ap-beijing.myqcloud.com
```

### 方案 3：Cloudflare Workers 代理 ⭐⭐⭐

**原理**：通过 Cloudflare Workers 创建一个代理，将请求转发到 Vercel

**优势**：免费、Cloudflare 在国内访问较好

**步骤**：

#### 3.1 注册 Cloudflare
```
1. 访问 https://dash.cloudflare.com
2. 注册免费账号
```

#### 3.2 创建 Worker
```javascript
// 在 Cloudflare Workers 中创建以下代码
export default {
  async fetch(request) {
    const url = new URL(request.url);
    // 替换为你的 Vercel 地址
    const targetUrl = 'https://your-project.vercel.app' + url.pathname + url.search;

    const modifiedRequest = new Request(targetUrl, {
      method: request.method,
      headers: request.headers,
      body: request.body,
    });

    return fetch(modifiedRequest);
  }
}
```

#### 3.3 部署后访问
```
https://your-worker.your-subdomain.workers.dev
```

### 方案 4：Coding Pages ⭐⭐⭐

**优势**：腾讯云旗下，国内访问快，免费

#### 4.1 注册 Coding
```
1. 访问 https://coding.net
2. 使用微信/邮箱注册
```

#### 4.2 创建项目
```
1. 创建项目 → 静态网站
2. 上传 oss-deploy 文件夹内容
3. 启用 Pages 服务
```

---

### 方案 5：自定义域名 + 国内 CDN ⭐⭐⭐⭐

**如果您有域名**，这是最优方案：

#### 5.1 准备工作
```
1. 购买域名（阿里云/腾讯云都可以）
2. 将 Vercel 部署的地址绑定到自定义域名
```

#### 5.2 配置国内 CDN
```
1. 在阿里云/腾讯云开通 CDN
2. 添加加速域名
3. 源站设置为 Vercel 地址
4. 配置 CNAME
```

## 🚀 快速决策

| 如果您... | 推荐方案 |
|----------|---------|
| 没有域名，想要最稳定的方案 | **阿里云 OSS** |
| 想要免费方案 | **Cloudflare Workers** 或 **Coding Pages** |
| 有域名，追求最佳体验 | **自定义域名 + CDN** |

| 已经有腾讯云账号 | **腾讯云 COS** |

## ❓ 为什么之前阿里云/腾讯云无法显示？

可能的原因：

1. **没有开启「静态网站托管」**：直接访问 OSS 域名会下载文件
2. **访问了错误的 URL**：应该访问静态网站托管的专用域名
3. **CORS 配置问题**：需要配置跨域规则
4. **缓存问题**：需要清除 CDN 缓存
