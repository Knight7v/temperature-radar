# 空调热销区域雷达 - 商业化设计方案

**项目名称：** kongtiaoredian.cn（空调热点）
**设计日期：** 2026-05-12
**当前状态：** 等待域名审核

---

## 1. 项目概述

### 1.1 目标
将现有空调热销区域雷达系统转型为商业化 SaaS 产品，为空调经销商/零售商提供数据驱动的销售机会分析服务。

### 1.2 目标客户
- 主要客户：空调经销商、零售商
- 客户需求：了解各地区销售机会，优化库存和营销决策
- 客户规模：从小型零售店到大型连锁店

### 1.3 商业模式
**Freemium + 广告收入混合模式**
- 免费功能：基础数据展示、今日热销
- 付费功能：热销分析、7天预测
- 广告收入：顶部Banner + 右侧悬浮卡片

## 3. 价格策略

### 3.1 阶段1：MVP期（快速启动）
**混合模式：单次购买 + 订阅制**

| 功能 | 价格 | 说明 |
|------|------|------|
| 免费功能 | ¥0 | 今日热销、基础地图、TOP榜单 |
| 热销分析 | ¥29.9/次 | 单次购买，24小时有效 |
| 7天预测 | ¥19.9/月 | 订阅制，月度自动续费 |
| 新用户优惠 | ¥9.9/月 | 首次购买7天预测优惠价 |

### 3.2 阶段2：成长期（扩展功能）
**订阅制套餐模式**

| 套餐 | 价格 | 功能 |
|------|------|------|
| 基础版 | ¥19/月 | 热销分析、今日数据 |
| 专业版 | ¥49/月 | 热销分析 + 7天预测 + 历史数据导出 |
| 企业版 | ¥199/月 | 多账号 + API接口 + 定制报告 |

### 3.3 价格策略分析

#### 单次购买（29.9元）
**优势：** 低门槛、灵活、现金流快、适合季节性业务
**不足：** 收入不稳定、客户流失率高、运营成本高

#### 订阅制（19.9元/月）
**优势：** 收入稳定可预测、客户粘性强、生命周期价值高
**不足：** 获客成本高、取消订阅风险、淡季挑战

#### 混合模式
结合两者优势，让客户有选择，降低决策门槛

## 4. 用户认证系统

### 4.1 认证方式
**双登录模式：手机号 + 微信扫码**

1. **手机号登录**
   - 手机号 + 验证码
   - 适合传统企业客户
   - 便于后续短信营销

2. **微信扫码登录**
   - 微信公众号/小程序授权
   - 降低注册门槛
   - 便于微信生态集成

### 4.2 用户权限体系
```
游客 → 免费用户 → 付费用户
  ↓        ↓          ↓
基础功能  热销分析   7天预测
        (单次)     (订阅)
```

### 4.3 实现技术
- **前端：** 表单验证、OAuth流程
- **后端：** Vercel Serverless Functions
- **存储：** Vercel KV（会话）、JSON文件（用户数据）
- **安全：** JWT Token、HTTPS、验证码限流

## 5. 支付系统

### 5.1 支付方式
- **微信支付** - 国内最主流
- **支付宝** - 企业用户常用

### 5.2 支付集成方案
**阶段1（MVP）：第三方聚合支付平台**
- 推荐：Payjs、虎皮椒
- 优势：接入简单、费率2-3%、支持双渠道
- 时间：1-2天完成集成

**阶段2（成长期）：直接对接官方API**
- 微信支付官方API
- 支付宝官方API
- 优势：费率更低、功能更丰富

### 5.3 订单流程
```
1. 用户选择付费功能
2. 系统生成订单
3. 跳转支付页面
4. 完成支付
5. 系统授权访问权限
6. 发送支付确认通知
```

### 7.3 权限控制
```javascript
// 权限检查逻辑
function checkPermission(feature, user) {
  const permissions = {
    'today-hotspot': true,  // 所有用户
    'heatmap': true,         // 所有用户
    'top-cities': true,      // 所有用户
    'hot-analysis': false,   // 需要付费
    'forecast-7d': false     // 需要订阅
  };

  if (permissions[feature]) return true;
  return user.hasAccess(feature);
}
```

## 9. 风险和挑战

### 9.1 技术风险
- **风险：** Vercel免费版限制
- **应对：** 监控使用量，及时升级

- **风险：** 支付系统安全
- **应对：** 使用成熟的第三方平台，遵循安全最佳实践

### 9.2 商业风险
- **风险：** 客户付费意愿低
- **应对：** 提供免费试用，收集反馈优化产品

- **风险：** 季节性波动
- **应对：** 混合定价模式，淡季提供促销活动

### 9.3 合规风险
- **风险：** 预付卡监管
- **应对：** 避免预充值模式，采用即付即用

- **风险：** 数据隐私
- **应对：** 遵守GDPR和国内数据保护法规
