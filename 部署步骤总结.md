## 📋 现在只需要三步：

### 第一步：创建 Gitee 仓库（2 分钟）

1. 打开 https://gitee.com
2. 注册/登录账号
3. 点击右上角 **「+」** → **「新建仓库」**
4. 填写：
   - 仓库名：`temperature-radar`
   - **选择「公开」** ⚠️ 必须公开才能启用 Pages
   - 其他不填
5. 点击 **「创建」**

---

### 第二步：上传文件（3 分钟）

1. 在仓库页面点击 **「文件」** → **「上传文件」**
2. 打开 Finder，前往：
   ```
   /Users/knight/Desktop/temperature-monitor-prototype/gitee-deploy
   ```
3. 拖拽这 4 个项目到上传区域：
   - `index.html` 文件
   - `css` 文件夹
   - `js` 文件夹
   - `assets` 文件夹
4. 点击 **「提交」**

---

### 第三步：启用 Gitee Pages（5 分钟 + 实名认证时间）

1. 点击顶部 **「服务」** → **「Gitee Pages」**
2. 如需实名认证，上传身份证（1-2 小时通过）
3. 认证后，选择：
   - 部署分支：`master`
   - 勾选 **「强制使用 HTTPS」**
4. 点击 **「启动」**
5. 等待 1-3 分钟部署完成
